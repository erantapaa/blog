Quaternions as rotations

Let $p$ be a 3d point represented as a quaternion using its homogeneous coordinates, $p = [w, \mathbf{v}]$ and let $q$ be any non-zero quaternion then

Theorem: The product $qpq^{-1}$ takes $p = [w, \mathbf{v}]$ to $p’ = [w, \mathbf{v’}]$

Before proving this theorem let’s make the following observation, we can express $q$ as a multiplication of a scalar quaternion times a unit quaternion i.e. $q = s\mathbf{U}q$ (where $s$ is a real quaternion and $\mathbf{U}q$ a unit quaternion) then $ qpq^{-1} = s\mathbf{U}qp(s\mathbf{U}q)^{-1} = s\mathbf{U}qp\mathbf{U}q^{-1}s^{-1} $, because the scalar multiplication is commutative $\mathbf{U}qp\mathbf{U}q^{-1}ss^{-1} = \mathbf{U}qp\mathbf{U}q^{-1}$, then without loss of generality we can say that that same formula applies when $q$ is a unit quaternion, finally notice that $\mathbf{U}q^{-1} = \mathbf{U}q^{*}$ so we can write the action as $qpq^*$ note that from now on, $q$ is assumed to be a unit quaternion without loss of generality

Next, let’s prove that the scalar part $qpq^{*}$ is the same as the scalar of $p$ (we can use the formula to find the scalar component of a quaternion)

Therefore the scalar part of $p$ remains constants in the operation i.e. if $p = [w, \mathbf{v}]$ then $p’ = qpq^{*} = [w, \mathbf{v’}]$, and because multiplication preserves norms then $\norm{p} = \norm{p’}$ and also $\norm{v} = \norm{v’}$ $\blacksquare$
